# StorybookAddon

## Description

Storybook addon to switch between differents css styles.

## Stack

- Angular app built with Nx
- Storybook v.6.0.21

## Install

Clone project : 

`git clone https://github.com/AvaNoilhetas/storybook-addon.git`

Init project : 

`npm install`

Start storybook :

`npm run docs:json && start-storybook -p 6006`
